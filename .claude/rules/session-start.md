# Session Start — Leitura Obrigatória

Ao iniciar qualquer sessão neste projeto, leia obrigatoriamente antes de qualquer resposta:

1. `CONTEXT.md` — estado atual do produto, pendências críticas, URLs de produção
2. `docs/fullstack-architecture.md` — stack e decisões técnicas (se existir)
3. `.claude/session-log.md` — tarefas pendentes da sessão anterior

Sem essa leitura, você não tem contexto suficiente para responder com precisão.
Nunca assuma que o projeto está em estado inicial — ele está em produção.
